#[repr(transparent)]
pub struct Fr(_);
Expand description

Represents an element of the scalar field $\mathbb{F}_q$ of the BLS12-381 elliptic curve construction.

The inner representation blst_fr is stored in Montgomery form as little-endian u64 limbs.
